# 宝宝日常记录小程序

一个轻量的微信小程序原型，用来快速记录宝宝的日常：

- 喂奶
- 换尿布
- 大小便
- 擦身体乳/喂药
- 洗澡
- 睡眠
- 外出活动
- 第一次和特别瞬间

## 当前版本

已完成一个可运行的初版结构：

- 快速记录页
- 时间线页
- 特别瞬间页
- 基础统计页
- 本地存储（wx.setStorageSync）

## 如何使用

1. 用微信开发者工具打开 `baby-journal-miniapp`
2. 选择小程序项目导入
3. 先用游客 appid 或替换成你自己的 appid
4. 直接预览和继续迭代
