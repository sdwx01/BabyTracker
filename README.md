# BabyTracker

微信宝宝喂养记录小程序，当前已支持：

- 首次进入欢迎页，可选择“添加宝宝开始记录”或“通过邀请码加入家庭”
- 首页快捷记录、时间轴、记录编辑
- 统计页、里程碑、家庭共享与设置
- 本地存储 + 自动云端同步
- 家庭邀请码协作、成员身份设置
- 里程碑媒体上传云存储、预览与删除回收

## 云开发接入步骤

1. 在微信云开发控制台创建环境。
2. 把 `utils/config.js` 中的 `CLOUD_ENV_ID` 改成你的环境 ID。
3. 在微信开发者工具中右键 `cloudfunctions/babyTracker`，执行“上传并部署：云端安装依赖”。
4. 在云开发数据库中创建集合 `bt_families` 和 `bt_members`。
5. 重新编译小程序。
6. 首次进入时按欢迎页引导创建宝宝，或使用邀请码加入家庭。
7. 在“我的”页开启云端同步后，记录会自动同步；家庭协作能力在“家庭管理”页查看。

## 当前云同步策略

- 小程序页面仍以本地缓存为直接数据源，保证页面响应速度。
- 应用启动时会尝试从云端拉取并覆盖本地缓存。
- 本地写操作后会自动异步上传整份家庭数据到云端。
- 如果云端失败，会自动回退到本地模式，并在“我的”页展示简短状态提示。

## 当前限制

- 家庭协作已通过邀请码共享家庭数据，成员映射保存在 `bt_members` 集合中。
- 目前使用整份 store 同步，适合 MVP；后续可拆成更细粒度的记录、里程碑等集合。
